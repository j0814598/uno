# BACKLOG

このアプリはまず「こうしき UNO ルール」だけで動きます。
あとから「ローカルルール（ハウスルール）」をオプションでオン/オフできるようにする予定です。

## ローカルルール候補（おうちルール）

設定画面（あとで追加）でトグルできるようにする。

- [ ] **ドロー2のかさねがけ**：ドロー2を出されたら、自分もドロー2を持っていれば重ねて次の人にまわせる
- [ ] **+4のかさねがけ**：ドロー4を重ねて出せる（+2にも重ねられる派 / +4のみ重ね派 の選択）
- [ ] **+4へのチャレンジ**：「本当に同色なかった？」とチャレンジできる（公式ルールにもあるが、子ども向けはオフ推奨）
- [ ] **ジャンプイン（同じカードならいつでも割り込み可能）**
- [ ] **0と7の交換**：0を出したら全員手札を回転、7を出したら好きな相手と手札交換
- [ ] **数字スタック**：同じ数字なら色違いでも複数枚同時出し
- [ ] **ドロー後すぐプレイ**：ヤマ札をひいたカードがその場で出せるなら出して良い（公式ルール / 現在は自動でターン終了）
- [ ] **UNOコール忘れペナルティ枚数**：2枚 / 4枚を選択
- [ ] **手札の表示順**：色順 / 数字順 / 引いた順
- [ ] **AIのつよさ**：やさしい / ふつう / つよい
- [ ] **音とアニメーション**：オン/オフ
- [ ] **ふりがな表示モード**：すべて ひらがな / 漢字あり
