# UNO（おうち向け Web アプリ）

ブラウザだけで遊べる、ひとりプレイ用 UNO（あなた vs CPU）。
インストール不要・サーバー不要・通信不要・課金なし。

## あそびかた

`index.html` をブラウザで開くだけ。

- ローカルで開く: ファイルをダブルクリックして開く
- もしくは簡単なサーバーで開く（推奨）:

```bash
# Python が入っている場合
python3 -m http.server 8000
# → http://localhost:8000/ を開く
```

## ルール

「こうしき UNO ルール」（108枚デッキ、最初に7枚配る、各種アクションカード、ワイルド/+4、UNOコール）

## 構成

- `index.html` — 画面
- `style.css` — みためのスタイル
- `game.js` — ゲームエンジン（ルール、デッキ、ターン進行）
- `ai.js` — CPU の手選び（シンプルなルールベース、外部API不使用）
- `ui.js` — 描画とユーザー操作
- `BACKLOG.md` — ローカルルールなど、これから追加する機能のメモ

## CPU について

CPU はブラウザ内で動く JavaScript のシンプルなロジックです。
外部AI（ChatGPT/Claude等のAPI）は使っていないので、**料金は発生しません**。
詳細は `ai.js` を参照。
